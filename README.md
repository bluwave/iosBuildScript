iosBuildScript
==============

ios build scripts to automate ios builds from command line

i usually have a directory structure like this with my ios projects


	--
	|
	|-projectName (src files usually here)
 	|
 	|-projectName.xcodeproj
 	|
 	|-buildScripts
         	|
         	|-build.sh
         	|
         	|-provisions
                 	|
                 	|-provisionFile
